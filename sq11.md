# 2️⃣ Memory of list from iterator


`sys.getsizeof()` only measures the **list container**, not the **objects inside it**.

Example:


* the list structure
* not the millions of integers stored inside.

So the **actual memory is MUCH larger**, but `getsizeof` does not count nested objects.
